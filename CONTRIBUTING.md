# Contributing to GitHub Profile Badges

Thank you for considering contributing! 🎉

## How to Contribute

1. **Fork** this repository
2. **Create a branch**: `git checkout -b fix/my-update`
3. **Make your changes** to `README.md` or other files
4. **Commit**: `git commit -m "Describe what you changed"`
5. **Push**: `git push origin fix/my-update`
6. **Open a Pull Request** with a clear description

## What to Contribute

- New badges that GitHub adds
- Corrections to earning conditions
- Better step-by-step instructions
- Typo fixes
- New tips or tricks

## Guidelines

- Keep descriptions accurate and concise
- Test badge earning steps before submitting
- Use proper markdown formatting
- Be respectful and constructive

Thank you! 🛡️
